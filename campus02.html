Dualer Studiengang: Software Engineering & Management

Das duale Studium an der FH CAMPUS 02 verbindet theoretisches Wissen mit praktischer Erfahrung im Unternehmen. Pro Semester wechseln sich Theoriephasen an der Hochschule mit Praxisphasen im Betrieb ab.

Sommersemester (März – August) 2025

  • Webentwicklung (Grundlagen)
  • Datenbanken und SQL (MySQL)
  • Softwareprojekt (Teamarbeit: das Spiel "UNO")
  • Java Core
  • Java Intermediat
  • Java Advance
  • Datenbankenprogrammieren
  • Algorithmen und Datenstrukturen
  • Softwareprojekt_Modelle (Teamarbeit) – 📄 Präsentation anzeigen

Wintersemester (Oktober – Jänner)

  • Interaktive Systeme
  • Web II
  • Programmiersprachen
  • Datenbankprogrammieren II
  • Software Design

Weitere Inhalte und Vertiefungen folgen im Laufe des Studiums.
